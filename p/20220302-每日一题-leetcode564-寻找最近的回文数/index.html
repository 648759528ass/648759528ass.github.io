564. 寻找最近的回文数

题面

给定一个表示整数的字符串 n ，返回与它最近的回文整数（不包括自身）。如果不止一个，返回较小的那个。

“最近的”定义为两个整数差的绝对值最小。

解析

这题其实不值得hard的难度，整体来说只是代码比较难写，需要分类讨论较多情况而已

首先对于个整数字符串n

我们找出它的大于它的最近回文数与小于它的最近回文数，然后把两个回文数与n的差取绝对值比较即可。

对于大回文数与小回文数，我们首先考虑的是在不对n的上半部分增减数字的情况下构造一个回文数，若这个回文数小于n，则该回文数为小回文数。反之若大于n，则该回文数为大回文数。若等于n则必须增减数字来构造回文数。

首先我们先讨论大回文数的构造：

若要取得大于n的最小回文数，则必是对n的上半部分进行一个 +1 操作(贪心的思想)(因为数字回文所以下半部分不重要)，然后使用上半部分继续构造回文数

同时我们考虑一下进位的情况:

对于 9999 来说 +1 后 变为10099 构造后 10001,为9999的上界回文数。即只需要构造即可，不需要特别在意进位。

然后我们讨论一下小回文数的构造:

与大回文数构造的思想相同,若要取得小于n的最小回文数，则必是对n的上半部分进行一个 -1 操作(贪心的思想)(因为数字回文所以下半部分不重要)，然后使用上半部分继续构造回文数.

我们再来考虑一下进位的情况。这时候我们就需要区分长度奇偶了。

先来讨论长度为偶数，进位后为奇数的情况:

对于100000 来说 -1后变 99000 假如我们直接按上部回文构造的话就会变为 99099,显然 99999 才是100000 的下界回文数，所以我们需要将中间位也变为9，

换句话来说就是检测到头部为0，需要将 len/2 这个位置变为 9

再来讨论长度为奇数，进位后为偶数的情况:

对于 10000 来说 -1 后变为 9900, 构造后变为9999 是正确的答案，所以长度为奇数进位后为偶数的情况是不需要特殊构造的。

class Solution {
public:
    long long int find_up(string n){
        auto lln = parse(n);
        int len = n.size();
        string local_tmp = n;
        for(int i = 0;i<len/2;i++){
            local_tmp[len-i-1] = local_tmp[i];  
        }
        auto local_n = parse(local_tmp);
        if(local_n>lln) return local_n;
        if(len%2){
            local_tmp[len/2]+=1;
            if(local_tmp[len/2] == '0'+10){
                for(int i = len/2;i>=0;i--){
                    if(i == 0 && local_tmp[i] == '0'+10){
                        local_tmp[i] = '0';
                        local_tmp = "1"+local_tmp;
                        break;
                    }else if(local_tmp[i] == '0'+10){
                        local_tmp[i-1]+=1;
                        local_tmp[i] = '0';
                    }else{
                        break;
                    }
                }
            }
        }else{
            local_tmp[len/2-1]+=1;
            if(local_tmp[len/2-1] == '0'+10){
                for(int i = len/2 - 1;i>=0;i--){
                    if(i == 0 && local_tmp[i] == '0'+10){
                        local_tmp[i] = '0';
                        local_tmp = "1"+local_tmp;
                        break;
                    }else if(local_tmp[i] == '0'+10){
                        local_tmp[i-1]+=1;
                        local_tmp[i] = '0';
                    }else{
                        break;
                    }
                }
            }
        }
        for(int i = 0;i<local_tmp.size()/2;i++){
            local_tmp[local_tmp.size()-i-1] = local_tmp[i];
        }
        return parse(local_tmp);
    }
    long long int find_low(string n){
        auto lln = parse(n);
        int len = n.size();
        string local_tmp = n;
        for(int i = 0;i<len/2;i++){
            local_tmp[len-i-1] = local_tmp[i];  
        }
        auto local_n = parse(local_tmp);
        if(local_n<lln) return local_n;
        if(len%2){
            local_tmp[len/2]-=1;
            if(local_tmp[len/2] == '0'-1){
                for(int i = len/2;i>=0;i--){
                    if(i == 0 && local_tmp[i] == '0'){
                        if(local_tmp.size() == 1) break;
                        local_tmp = local_tmp.substr(1);
                        break;
                    }else if(local_tmp[i] == '0'-1){
                        local_tmp[i-1]-=1;
                        local_tmp[i] = '9';
                    }else{
                        break;
                    }
                }
            }
        }else{
            local_tmp[len/2-1]-=1;
            if(local_tmp[len/2-1] == '0'-1||(len/2-1 == 0 && local_tmp[len/2-1] == '0')){
                for(int i = len/2 - 1;i>=0;i--){
                    if(i == 0 && local_tmp[i] == '0'){
                        local_tmp[len/2] = '9';
                        local_tmp = local_tmp.substr(1);
                        break;
                    }else if(local_tmp[i] == '0'-1){
                        local_tmp[i-1]-=1;
                        local_tmp[i] = '9';
                    }
                }
            }
        }
        for(int i = 0;i<local_tmp.size()/2;i++){
            local_tmp[local_tmp.size()-i-1] = local_tmp[i];
        }
        return parse(local_tmp);
    }
    long long int parse(string n){
        long long int ret = 0;
        for(int i = 0;i<n.size();i++){
            ret*=10;
            ret += n[i]-'0';
        }
        return ret;
    }
    string nearestPalindromic(string n) {
        auto num = parse(n);
        auto low = find_low(n);
        auto high = find_up(n);
        if(num-low<= high - num){
            return to_string(low);
        }else{
            return to_string(high);
        }
    }
};
