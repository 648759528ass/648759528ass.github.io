504. 七进制数

题面

给定一个整数 num，将其转化为 7 进制，并以字符串形式输出。

解析

用十进制转任意进制的方法来解即可，首先解决正负问题

class Solution {
public:
    string convertToBase7(int num) {
        if(num == 0) return "0";
        string pre = num<0?"-":"";
        num = abs(num);
        string ans = "";
        while(num){
            ans=to_string(num%7) + ans;
            num/=7;
        }
        return pre+ans;
    }
};
